# 眉毛サロン比較メディア 店舗情報AI自動更新システム

眉毛サロン比較メディア（400記事以上）の店舗情報を、ホットペッパービューティーを正として
AIで自動収集・統一フォーマット化・差分検知するためのプロジェクトです。

## ドキュメント

- [要件定義.md](./要件定義.md)
- [設計書.md](./設計書.md)
- クライアント説明用ページ: `index.html`（GitHub Pagesで公開）

## ツール構成

- **ツールA**: ホットペッパービューティーから店舗情報を取得し、Googleスプレッドシートに店舗マスタを作成する
- **ツールA'**: 既存記事内の店舗情報表を、店舗マスタを参照して統一フォーマットへ置換する
- **ツールB**: ホットペッパービューティーを定期巡回し、店舗マスタとの差分を検知・記録する

詳細は [設計書.md](./設計書.md) を参照してください。
